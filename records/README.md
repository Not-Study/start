# 统计记录

每日学习打卡统计数据，由 GitHub Actions 自动生成。

## 目录结构

```
records/
├── 2025-11-04/
│   ├── README.md              # 当日统计报告
│   └── not-study-*.json       # 详细提交记录
├── 2025-11-05/
└── ...
```

## 统计说明

- **运行时间**: 每天北京时间 00:00
- **统计范围**: 所有 `not-study-*` 仓库
- **数据格式**: Markdown 报告 + JSON 详情

## 查看数据

直接浏览对应日期的文件夹即可。
